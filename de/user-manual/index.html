    # Hauptfenster

    Nach dem Start der Anwendung wird der Hauptbildschirm angezeigt. Alle Operationen werden von diesem Bildschirm aus über das Hauptmenü, die linken unteren Tasten oder die Tasten auf der rechten Seite gestartet.

    # Baum

    Der Anwendungsbaum dient dazu, Benutzerdaten auf eine einfach zu verwaltende Weise zu organisieren. Die Hierarchie der Elemente besteht aus dem Stammarbeitsbereich, Projekten, Platinen und Komponenten.

    Baumelemente folgen diesen Regeln:

    • Ein Arbeitsbereich kann mehrere Projekte enthalten.
    • Ein Projekt kann mehrere Leiterplatten unterschiedlicher Art enthalten.
    • Eine Leiterplatte kann mehrere Komponenten unterschiedlicher Art enthalten.
    • Es kann jeweils nur ein Arbeitsbereich in der Anwendung geöffnet sein, aber es ist möglich, zwischen den Arbeitsbereichen zu wechseln. (siehe Arbeitsbereich Abschnitt)

    # Schnelltasten

    Schnelltasten werden in der linken unteren Seite des Anwendungsfensters angezeigt. Sie werden verwendet, um Hinzufüge-, Duplizierungs- und Löschvorgänge im Baum durchzuführen.

    • Die Hinzufügen Taste wird mit einem + Zeichen dargestellt.
    • Die Duplizieren Taste wird mit einem Kopiere Datei Zeichen dargestellt.
    • Die Schaltfläche Löschen wird durch einen Papierkorb dargestellt

    # Verwendung von Schnelltasten

    • Wählen Sie ein Projekt, eine Leiterplatte oder ein Komponentenelement im Baum aus.
    • Klicken Sie auf die Schnelltast der gewünschten Aktion; die Aktion wird für das ausgewählte Baumelement ausgeführt.

    # Statusleiste

    Die Statusleiste ist die Fußzeile der Anwendung. Sie wird verwendet, um Informationen wie den Status der Geräteverbindung anzuzeigen. Wenn das Gerät angeschlossen ist, wird die Meldung "Mit Gerät verbunden" mit der Geräte-Firmware-ID in Klammern angezeigt.

    # Arbeitsbereiche

    Ein Arbeitsbereich ist eine Sammlung von verwandten Projekten und deren Komponenten. Es ist möglich, einen bestehenden Arbeitsbereich zu laden und zu bearbeiten oder einen neuen zu erstellen.

    # Arbeitsbereich erstellen

    Schritte zur Erstellung des Arbeitsbereichs

    • Wählen Sie im Menü Arbeitsbereich die Option Neu
    • Geben Sie den Namen ein
    • Speicherplatz des Arbeitsplatzes auf der Festplatte auswählen
      • Standardmäßig schlägt die Anwendung vor, den Arbeitsbereich im Pfad Users\Public\Documents\Martin\EasySolder\Workspaces zu speichern. Der Pfad kann mit der Schaltfläche Durchsuchen geändert werden.
    • Klicken Sie auf Erstellen. Der neue Arbeitsbereich wird auf der Festplatte erstellt und in die Anwendung geladen.

    # Freier Arbeitsbereich

    Ein freier Arbeitsbereich ist über das Menü Arbeitsbereich erreichbar und öffnet einen Dateibrowser mit dem zuletzt gespeicherten Pfad.

    Schritte zu einem freien Arbeitsbereich

    • Menü Arbeitsbereich öffnen
    • Wählen Sie Öffnen
    • Suchen Sie im Dateibrowser nach dem gewünschten Arbeitsbereich.
      • Es ist wichtig, den Arbeitsbereichsordner beim Öffnen auszuwählen.
    • Wählen Sie die Datei BevorzugterArbeitsbereichsname.json aus dem Ordner aus
    • Wählen Sie Öffnen

    # Neuester Arbeitsbereich

    Öffnet einen kürzlich geschlossenen Arbeitsbereich. Verwenden Sie dies, um schnell zwischen den Arbeitsbereichen zu wechseln. Der vorher laufende Arbeitsbereich wird automatisch geschlossen, bevor der neue geöffnet wird. Die Anwendung speichert bis zu 5 zuletzt genutzte Arbeitsbereiche.

    Schritte zum Öffnen des neuesten Arbeitsbereichs

    • Menü Arbeitsbereich öffnen
    • Wählen Sie Neueste
    • Wählen Sie einen der aufgelisteten Arbeitsbereiche aus.

    • Speichern

    # Leiterplatte

    Die Leiterplatte repräsentiert das Board und ist in Projekten enthalten.
    Die Leiterplattendaten setzen sich wie folgt zusammen:

    • Name der Leiterplatte
    • Löttyp: Kann bleifrei oder verbleit sein.
    • Leiterplattenbild: Ermöglicht ein einfaches visuelles Erkennen der Leiterplatte.
    • Typ der Unterheizung: Muss entsprechend der installierten Unterheizung ausgewählt werden. Die folgenden Unterheizungen werden derzeit unterstützt:
      • IXH
        • Nur für Expert 04.6 Rework Station
      • HV
        • Anzahl der Lampen: 6
        • Kann für die Expert 10.6 Rework Station verwendet werden
      • HXV
        • Anzahl der Lampen: 8
        • Kann für die Expert 10.6 Rework Station verwendet werden

    Beim Hinzufügen einer neuen Leiterplatte muss ein Unterheizungstyp ausgewählt werden. Wenn die Anwendung an eine Rework Station angeschlossen ist, wird die erkannte Unterheizung automatisch ausgewählt. Basierend auf der gewählten Unterheizung wird ein Standardprofil der Unterheizung erzeugt.

    Profil der Unterheizung

    Das Unterheizungsprofil dient dazu, die gewünschte Leiterplattentemperatur zu erreichen und die Leiterplatte beim Löten gleichmäßig zu erwärmen, um Schäden durch große Temperaturunterschiede (Verzug, Risse, Biegungen, etc.) zu vermeiden.

    Das generierte Unterheizungsprofil enthält nur minimale Heizeinstellungen und muss in der Konfiguration Unterheizungsprofil bearbeitet werden oder mit dem Unterheizung Autoprofiler, um die gewünschte Temperatur auf der Leiterplatte zu erreichen

    Jede Unterheizung hat eine andere Möglichkeit, das Profil zu konfigurieren. Wenn Sie daher die Unterheizung auf der Leiterplatte wechseln, wird ihr Unterheizungsprofil auf den Standardwert zurückgesetzt

    Das Fenster zur "Konfiguration Unterheizungsprofil" kann über die Schaltfläche Konfigurieren für die ausgewählte Leiterplatte geöffnet werden. Die angezeigten Eingänge sind abhängig vom konfigurierten Unterheizungstyp.

    # HV und HXV Unterheizungsprofile

    Für HV und HXV Unterheizungen können die folgenden Einstellungen vorgenommen werden:

    Name der Einstellung Details
    Lampen Die Lampen können durch Verwendung der angezeigten Kontrollkästchen angepasst werden. Wenn eine Rework Station angeschlossen ist, werden die von der Handablage abgedeckten Leuchten hervorgehoben.

    # Betriebsart: Temperatur

    In dieser Betriebsart muss ein Sensor an der Leiterplatte angebracht werden. Sensor 1 oder 5 kann verwendet werden. Wenn beide Sensoren angeschlossen sind, wird bei der Probenahme der höhere Temperaturwert berücksichtigt.

    Hinweis

    Die Betriebsart Temperatur entspricht dem geschlossenen Regelkreis in früheren Softwareversionen.

    Die Betriebsart Temperatur verwendet die folgenden Einstellungen:

    Name der Einstellung Details
    Leiterplattentemperatur Dies ist die gewünschte Leiterplattentemperatur in °C. Die Unterheizung startet mit voller Leistung und überwacht, wenn die Temperatur am Sensor steigt. Wenn die gewünschte Temperatur am Sensor erreicht ist, geht die Unterheizung in den Regelbetrieb. Anhand der ständig überwachten Sensordaten schaltet er sich bei Bedarf ein und aus, um die gewünschte Leiterplattentemperatur zu halten.
    Mindestwert: 50
    Maximalwert: 210
    Standardwert: 150
    Vortemperatur Vorheiztemperatur stellt die Leiterplattentemperatur dar, die von der Unterheizung erreicht werden muss, bevor das Profil der Oberheizung gestartet wird. Wenn dieser Parameter einen Wert größer als 0 hat, werden die Profile Unterheizung und Oberheizung nicht gleichzeitig gestartet.
    Mindestwert: 0
    Maximalwert: 200
    Standardwert: 0

    # Betriebsart: Leistung

    In dieser Betriebsart kann die Leiterplatte ohne Anbringung von Sensoren auf die gewünschte Temperatur erwärmt werden. Die Unterheizung wird nach einem zuvor erlernten oder direkt eingegebenen Parametern betrieben.

    Hinweis

    Die Betriebsart Leistung entspricht dem offenen Regelkreis in früheren Softwareversionen.

    Optionale Sensoren

    Im Leistungsbetrieb können Sensoren an die Anschlüsse der Rework Station angeschlossen werden. In diesem Fall werden Temperaturkurven auf dem Grafikdisplay beobachtet, die jedoch keinen Einfluss auf die Prozesssteuerung haben.

    Sensorplatzierung

    Ein Temperatursensor auf der Unterseite der Leiterplatte wird durch die Wärmezufuhr der Oberheizung beeinflusst. Es ist normal, dass ein Temperaturanstieg beobachtet wird; dies wird in der Regel durch die falsche Platzierung des Sensors verursacht.

    Die Unterheizung erreicht die gewünschte Leiterplattentemperatur, indem sie ein Leistungsprofil anwendet, das aus vier Phasen besteht: ansteigen, pausieren, halten und abkühlen.

    Phase Beschreibung
    Anstiegsphase Es wird eine Leistung eingestellt (in der Regel 100% Leistung), die die Leiterplatte für eine definierte Zeitspanne erwärmt.
    Pausenphase Nach der Aufheizphase folgt eine Pause bei Abschalten der Unterheizung, die aber weiterhin ihre Restwärme überträgt. Während dieser Phase steigt die Temperatur der Leiterplatte weiter an.
    Haltephase In dieser Phase startet die Unterheizung wieder und schaltet eine Leistung ein, die die Temperatur der Leiterplatte konstant hält.
    Abkühlphase Die Abkühlphase vervollständigt das Nacharbeitsprofil.

    Die Parameter, die das Leistungsprofil definieren, werden im Folgenden beschrieben. Mit dem Profiler können die meisten dieser Werte automatisch berechnet werden. Der einzige Parameter, der vom Profiler nicht berechnet wird, ist die Leistungsabfallrate.

    Parametername Wert
    Leiterplattentemperatur Gewünschte Leiterplattentemperatur. Dieser Wert ist schreibgeschützt. Er kann im Profiler Fenster geändert werden, indem der Profiler ausgeführt wird.
    Vorlaufzeit Gibt die Zeitverzögerung in Sekunden vom Start der Unterheizung bis zum Start der Oberheizung an. Wenn dieser Wert 0 ist starten Unterheizung und Oberheizung gleichzeitig. Dies ist das Zeitäquivalent des Vor-temp. Werts aus der Betriebsart Temperatur.
    Mindestwert: 0
    Maximalwert: 500
    Steigfähigkeit Leistungsniveau, das während der Anstiegsphase aufrechterhalten wird
    Dieser Wert wird in % angegeben. In der Regel wird die Anstiegsleistung mit 100% verwendet, um die gewünschte Temperatur in kürzester Zeit zu erreichen.
    Mindestwert: 0
    Maximalwert: 100
    Anstiegszeit Zeitintervall, in dem die Anstiegsleistung aufrechterhalten wird
    Mindestwert: 1
    Maximalwert: 500
    Pausenzeit Wert in Sekunden, der die Dauer der Pausenphase definiert.
    Die Pausenzeit muss als Sicherheitsmaßnahme eingestellt werden, da eine schnelle Erwärmung ohne kurze Unterbrechung des Energieeinsatzes zu einer Temperaturüberschreitung führen würde
    Mindestwert: 0
    Maximalwert: 255
    Halteleistung Definiert die Leistungsstufe, die während der Haltephase aufrechterhalten wird. Wie der Wert der Anstiegsleistung wird er als Prozentsatz der maximalen Unterheizleistung ausgedrückt.
    Mindestwert: 0
    Maximalwert: 100
    Leistungsabfallrate Die Leistungsabfallrate reduziert die Halteleistung im Laufe der Zeit um den angegebenen Wert. Sie wird in Prozent pro Sekunde ausgedrückt.
    Die Notwendigkeit dieses Parameters ergibt sich aus der Möglichkeit, dass durch die Energiezufuhr aus der Oberheizung die Temperatur der Leiterplatte/Modul kontinuierlich steigen kann und keine Halteleistungseinstellung bestimmt werden kann, die eine konstante Leiterplattentemperatur gewährleisten würde.
    Mindestwert: 0.00
    Maximalwert: 10.00

    Beispiel für einen Leistungsabfall

    Für die Leistungsabfallrate ist eine Einstellung von 0.05% ein sinnvoller Wert, d.h. die Leistungseinstellung in der Haltephase wird um 0.05%/s reduziert, d.h. 1% in 20 Sekunden bis zum Ende des Profils.

    # IXH Unterheizungsprofile

    Bei der IXH-Unterheizung besteht das Profil nur aus einer Anstiegsphase, direkt gefolgt von einer Haltephase. Die folgenden Einstellungen definieren das Profil der Unterheizung:

    Einstellung Beschreibung
    Vorwärmzeit Gibt die Zeitverzögerung in Sekunden vom Start der Unterheizung bis zum Start der Oberheizung an. Es hat die gleiche Bedeutung wie in den HV und HXV-Profilen.
    Startleistung Leistung, die zu Beginn der Anstiegsphase angewendet wird. Diese wird allmählich ansteigen, um den Endleistungswert im Laufe des Anstiegsintervalls zu erreichen. Der Wert wird in Watt angegeben.
    Endleistung Leistung, die am Ende der Anstiegsphase angewendet wird. Der Wert ist in Watt angegeben.
    Halteleistung Leistungsstufe, die während der Haltephase angelegt wird. Der Wert ist in Watt angegeben, und bleibt während der Haltephase konstant.

    # Unterheizungs-Profiler

    Diese Funktion ist für alle Unterheizungstypen verfügbar. Damit werden die richtigen Heizparameter für das Unterheizungsprofil ermittelt.

    Solltemperatur und Vorwärmtemperatur werden als Eingangsgrößen für den Profiler-Prozess verwendet. Der Profiler berechnet den Rest der Prozessparameter.

    Die maximal einstellbare Temperatur für die Unterheizung beträgt 210°C, dies dient auch als Schutz für das Bauteil. Überhitzung kann zu Schäden an der Leiterplatte führen.

    Der Unterheizungs-Profiler kann durch Drücken der Schaltfläche Ausführen Profiler gestartet werden. Vor Beginn des eigentlichen Prozesses wird der Bediener aufgefordert, eine Reihe von Anforderungen zu bestätigen.

    # Anforderungen an den Unterheizungs-Profiler

    Anforderungen Beschreibung
    Sensor anschließen Der Unterheizungssensor muss an der Unterseite der Leiterplatte angeschlossen werden.
    Für Expert 10.6 Rework Stations, kann Sensor 1 oder Sensor 5 verwendet werden. Wenn beide Sensoren angeschlossen sind, wird bei der Probenahme der höhere Temperaturwert berücksichtigt.
    Für Expert 04.6 Rework Stations kann Sensor 1 verwendet werden.
    Leiterplattentemperatur prüfen Die Temperatur an dem Leiterplattensensor muss unter 45°C liegen. Wenn die Sensortemperatur höher als 45°C ist, starten die Rework Stationen des Expert 10.6 automatisch die Kühlung mit Lüftern, während die Rework Stationen des Expert 04.6 nur darauf warten, dass die Temperatur sinkt. In allen Fällen kann dieser Kühlungsschritt durch Drücken der Starttaste umgangen werden.

    Schritte zum Ausführen des Profilers

    • Verbinden Sie Sensor 1 oder 5 mit der Leiterplatte
      • Für das Gerät Expert 04.6 steht nur Sensor 1 zur Verfügung.
      • Es wird empfohlen, den Sensor an einer dunkleren Stelle der Leiterplatte zu platziere, um glänzende Oberflächen zu vermeiden.
      • Wenn beide Sensoren angeschlossen sind, berücksichtigt die Anwendung das Maximum zwischen den beiden Sensoren.
    • Leiterplatte auswählen
    • Wählen Sie Konfigurieren
    • Klicken Sie auf Profiler
    • Wählen Sie Temperatur und Vorheiztemperatur
    • Profil ausführen
      • Die Anwendung erinnert daran, den Sensor beim Start des Profillaufs anzubringen.
    • Schließen Sie das Profiler Fenster
    • Denken Sie daran, die Einstellungen im Fenster Konfiguration Unterheizungsprofil zu speichern

    # Betrieb des Unterheizungsprofilers

    Wenn der Unterheizungs-Profiler startet, werden die Lampen eingeschaltet. Der Profiler versucht, die Solltemperatur so schnell wie möglich mit 100% der verfügbaren Leistung zu erreichen. Nach diesem Zeitpunkt steuert die Software die Stromversorgung, so dass die Baugruppe auf der gewünschten Temperatur bleibt. Der Profiler analysiert die Steuerungen der Unterheizung und ermittelt einen optimalen Leistungswert (Halteleistung), Anstiegszeit und Pausenzeit.

    # Komponentenprofil

    Ein Bauteilprofil enthält die Informationen, die für das Löten und Entlöten benötigt werden. Die Profilparameter werden je nach Bauteiltyp und Leiterplattentyp spezifisch angepasst.

    • Einem Oberheizungsprofil
    • Aufnahmedaten

    Zukünftige Versionen

    Eine Änderung des Unterheizungsprofils auf Leiterplattenebene ändert nichts an den bestehenden Komponenten der Leiterplatte, aber neue Komponenten verwenden das aktualisierte Unterheizungsprofil.

    Das Oberheizungsprofil wird basierend auf der gewählten Lötdüse erstellt. Ein Wechsel der Lötdüse eines Bauteils ändert nichts an bestehenden Profilen.

    # Oberheizung Autoprofiler

    Verwenden Sie den Autoprofiler der Oberheizung, um die erzeugten Richtwerte für das Profil der Oberheizung weiter zu verbessern. Dies gewährleistet ein gutes Reflow-Profil für das Löten und Entlöten. Der Autoprofiler kann die Haltetemperatur und die Spitzenzeit (falls erforderlich) für jede Phase der Oberheizung verfeinern.

    Zusätzliche Informationen

    Das Easy Solder-Profil steuert den Energiefluss, um sicherzustellen, dass die thermischen Grenzwerte für Komponenten, andere umgebende Geräte und die Leiterplatten nicht überschritten werden. Die Werte für das erste Profilerverfahren werden mit dem ausgewählten Lötwerkzeug geladen. Daher ist es empfehlenswert, SMD-Daten nicht zu ändern, bevor Sie den Profiler einmal mit den Originaleinstellungen ausgeführt haben. Normalerweise wird zum Entlöten der Oberheizung ein Profiler verwendet, da dies der einfachste Weg ist, den Schmelzpunkt für Chipkontakte zu bestimmen.

    # Betrieb des Autoprofilers

    1. Autoprofiler starten
      Klicken Sie auf der Seite Komponentenprofil auf die Schaltfläche Autoprofiler ausführen. Dadurch öffnet sich das Fenster Lötvorbereitung das sich um alle Vorlötkontrollen kümmert.
    2. Bestätigen Sie die Vorlötkontrollen im Fenster Lötvorbereitung:
      1. Leiterplatteneinrichtung prüfen, Sensor anschließen
        Schließen Sie bei den Expert 10.6 Rework Stationen den Sensor 6 an die Komponente an. Für Expert 04.6 Rework Stationen müssen Sie Sensor 2 anschließen. In beiden Fällen wird empfohlen, den Sensor möglichst an die Oberseite des Bauteils anzuschließen.
      2. Leiterplattentemperatur prüfen
        Bei Expert 10.6 Rework Stationen beginnen die Lüfter automatisch mit der Kühlung der Leiterplatte. Die Anwendung wartet eine voreingestellte Zeit, bevor sie zur nächsten Phase übergeht, um sicherzustellen, dass die Temperatur der Leiterplatte unter 45°C liegt. Alternativ können Sie auch die Kühlung überspringen und mit dem nächsten Schritt fortfahren.
      3. Lötdüsenversatz prüfen
        An dieser Stelle senkt sich die Lötdüse über das Bauteil. Mit den Pfeiltasten können Sie kleine Korrekturen an der Ausrichtung der Lötdüse vornehmen. Die Tasten Große Bewegungen treiben den Arm über einen Abstand von 1 mm an. Die Tasten Kleine Bewegungen treiben den Arm über einen Abstand von 0.2 mm an.
        Versatz speichern: Speichert die angewandten Korrekturen im Komponentenprofil auf der Festplatte.
    3. **Drücken Sie die Taste Start ** um mit dem Erwärmen der Leiterplatte und der Komponente zu beginnen. Die folgenden Schritte werden automatisch ausgeführt:
      1. Die Oberheizung führt die Anstiegsteilphase, wie sie durch die Parameter Anstiegsrampe und Endtemperatur definiert ist, durch. Endtemperaturparameter.
      2. Am Ende der Aufstiegsteilphase:
        • Der Profiler überprüft die Temperatur am Sensor 6. Liegt sie unter dem Wert der Bauteiltemperatur, wird die Spitzenzeit automatisch berechnet.
        • IWenn dies die letzte Lötphase ist, werden die Tasten Schmelzpunkt und Stopp verfügbar.
      3. Wenn die Spitzenzeit ungleich 0 ist, wird die Endtemperatur für die Dauer der Spitzenzeit gehalten.
      4. Die Oberheizung wechselt in den Regelmodus. In diesem Modus verwendet sie Informationen des Sensors und passt die Temperatur im Lötgriffel ständig an, um die gewünschte Temperatur am Bauteil zu erreichen. Wenn mehr Phasen ausgeführt werden müssen, ändert der Profiler die Haltezeit für die aktuelle Phase nicht.
    4. **Drücken Sie die Taste Schmelzpunkt ** sobald sich die Temperatur am Lötgriffel auf einen konstanten Wert eingestellt hat. Der Profiler läuft noch bis zu 20 Sekunden, nachdem der Knopf gedrückt wurde weiter, um sicherzustellen, dass die Pads gleichmäßig geschmolzen werden.
      -- ODER --
      Betätigen Sie die Stopp Taste, um die Lötphase sofort zu stoppen und aktualisieren Sie die Haltezeit entsprechend.
    5. Aufnahme
      Die Aufnahme wird ausgeführt, wenn sie im Profil aktiviert ist. Verwenden Sie die Taste Vakuum aus, um das Vakuum nach der Aufnahme des Bauteils auszuschalten..
    6. Die Abkühlphase läuft, wenn sie in den Profildaten konfiguriert wurde.
    7. Das Fenster Profillaufstatistik wird am Ende des Profils angezeigt.

    Info

    Der Profiler berechnet die Haltetemperatur aus den Temperaturwerten des Lötgriffelfeedbacks während der letzten 20 Sekunden der Halteteilphase.

    # Arm-Kalibrierung

    Der Kalibriervorgang am 10.6. muss durchgeführt werden, bevor eine Platzierung durchgeführt werden kann. Der Zweck der Kalibrierroutine ist die Messung von Arm- und Kameraparametern. Da jedes Kameraobjektiv unterschiedliche Parameter hat, muss für jedes eine Kalibrierung durchgeführt werden.

    Um eine Arm-Kalibrierung durchzuführen:

    1. **Öffnen Sie das Fenster Arm-Kalibrierung ** aus dem Hauptfenster -> Hauptmenü -> Werkzeuge -> Arm-Kalibrierung
    2. Bereiten Sie das Gerät für die Kalibrierung vor.
      • Entfernen Sie die Leiterplatte vor der Kalibrierung
        Stellen Sie sicher, dass vor Beginn der Kalibrierung keine Leiterplatte über der Unterheizung installiert ist. Während des Prozesses wird die Düse abgesenkt und führt einige niedrige Bewegungen aus, die auf Komponenten auf der Leiterplatte treffen könnten.
      • Wenn Sie Flip-Chip-Linsen verwenden, installieren Sie die Mikro-SMD-Düse und stellen Sie sicher, dass das Kontrollkästchen Mikro-SMD-Düse aktiviert ist.
    3. Klicken Sie auf die Schaltfläche Kalibrierung starten.
      Der größte Teil des Prozesses ist automatisiert. Der einzige Schritt, der eine Benutzereingabe erfordert, ist die Ausrichtung der Kamera. Dieser Schritt wird nur für das BGA Objektiv durchgeführt.
      • Kameraausrichtung
        • Die Düse wird in die Mitte des Fahrbereichs bewegt und ein roter Kreis erscheint im Bild.
        • Verwenden Sie einen Inbusschlüssel, um die Kamera so einzustellen, dass sich der Blob innerhalb des roten Kreises befindet. Er muss nicht genau in der Mitte des Kreises liegen, irgendwo im Kreis ist in Ordnung.
        • Nach der Überprüfung, ob die Kamera richtig zentriert ist, drücken Sie die Taste Fortfahren
    4. Nach Abschluss der Kalibrierung können Sie das Fenster schließen.

    Ziele der Arm-Kalibrierung:

    • Stellt sicher, dass die Parameter für die Bildhelligkeit optimal sind.
    • Misst die Armbewegungsfähigkeiten und ordnet sie den realen Einheiten (Millimeter und Grad) zu.
    • Ermöglicht es dem Bediener, die Kameraausrichtung einzustellen.
    • Berechnet perspektivische Parameter, die eine genaue Bauteilplatzierung gewährleisten

    # Platzierung

    Verwenden Sie das Platzierungsverfahren, um BGA- und QFN-Komponenten an ihrem Zielort zu positionieren.

    Diese Funktion ist für Expert 10.6 Geräte mit AVP-Arm verfügbar. Der AVP-Arm kann die Komponente an ihrer Zielposition auf der Leiterplatte platzieren. Nach der Markierung der beiden Zielpositionen und der Komponente auf dem Kamerabild, wird der Platzierungsprozess automatisch durch den AVP-Arm ausgeführt.

    # BGA-Platzierungsverfahren

    Um die Platzierung mit einem BGA-Bauteil durchzuführen, führen Sie die folgenden Schritte aus:

    1. Stellen Sie sicher, dass die Software über Kalibrierparameter für das verwendete Objektiv verfügt. Wenn die Armkalibrierung zuvor durchgeführt wurde, können Sie das Fenster Platzierung direkt öffnen. Andernfalls zeigt die Software beim Versuch, das Fenster Platzierung zu öffnen, eine Meldung an, die eine Armkalibrierung erfordert. Führen Sie die Armkalibrierung gemäß den Schritten durch, die im Abschnitt Kalibrierung dieser Dokumentation beschrieben sind.
    2. Setzen Sie die Leiterplatte auf das Gerät auf.
    3. Wählen Sie die Komponente im Hauptfensterbaum aus.
    4. Öffnen Sie das Fenster des Komponentenprofils.
    5. Klicken Sie im Profilfenster auf Platzierung.
    6. Stellen Sie sicher, dass der Kippschalter für den Platzierungsmodus auf BGA eingestellt ist.
    7. Sie können den Schieberegler Zoom verwenden, um den Bild-Zoom einzustellen. Sie können den Bild-Zoom auch mit dem Mausrad anpassen, während Sie mit der Maus über das Bild fahren.
    8. Markieren Sie den Zielort für die Komponente.
      • Verwenden Sie die 3 Marker, um die Zielposition des Rechtecks auf der Leiterplatte zu definieren. Zwei der Marker definieren benachbarte Eckpunkte, und der dritte Marker definiert die gegenüberliegende Linie.
      • Klicken Sie mit der Maus auf die Marker und ziehen Sie sie an die gewünschte Stelle. Das Zielrechteck wird entsprechend aktualisiert.
    9. Wenn das Zielrechteck entsprechend markiert ist, klicken Sie die Vakuum an Taste. Die Platzierungsdüse bewegt sich in Richtung Bildmitte.
    10. Befestigen Sie das Bauteil an der Platzierungsdüse.
    11. Stellen Sie sicher, dass das Bauteil nicht zu stark gedreht ist.
      • Die Platzierungsdüse kann je nach Hardware ca. 10 Grad nach links oder rechts gedreht werden.
    12. Klicken Sie auf die Schaltfläche Komponente absenken. Die Platzierungsdüse sinkt näher an die Leiterplatte heran, so dass das Bauteil in den Fokus rückt.
    13. Markieren Sie das Bauteil mit den drei Platzierungsmarkern
      • Das Markierungsverfahren ähnelt dem Markieren der Zielposition auf der Leiterplatte.
    14. Wenn die Komponente ordnungsgemäß gekennzeichnet ist, klicken Sie auf Autoplace.
      • Die Düse dreht sich, um das Bauteil parallel zur Zielposition auszurichten.
      • Anschließend verfährt der Arm, um das Bauteil über die Zielposition zu bewegen.
      • Schließlich fährt die Düse nach unten, bis das Bauteil auf der Leiterplatte platziert ist, und schaltet das Vakuum ab, um das Bauteil zu lösen.

    Hinweis

    Nachdem Sie auf Komponente absenken geklickt haben, können Sie die Düsenhöhe anpassen, indem Sie auf die Schaltfläche Höhe ändern klicken. Dadurch wird das Fenster Höhe ändern nach oben gezogen, so dass die Düse nach oben und unten bewegt werden kann. Eine große Bewegung hat eine Länge von 0,5 mm. Eine kleine Bewegung hat eine Länge von 0,1 mm.

    # QFN-Platzierungsverfahren

Stattdessen wird die Bauteilhöhe manuell gesteuert, um sicherzustellen, dass die Lotpaste geschützt ist. Das Ausschalten des Vakuums beendet den QFN-Platzierungsvorgang.

    18. Klicken Sie auf die Schaltfläche Ende Platzierung.

    # Kamerakalibrierung

    Die Kamerakalibrierung ist nur für das Maxi BGA XL Objektiv erforderlich. Dieser Prozess legt einen Satz optischer Parameter (Brennweite, Hauptpunkt, Verzerrung) fest, die aufgrund der erhöhten Bildgröße während der Armkalibrierung nicht gemessen werden können.

    # Allgemeine Informationen

    • Der Vorgang ist bei Verbindung mit dem Gerät verfügbar.
    • Schachbretter werden im Kalibrierungsprozess verwendet.
    • Das Maxi BGA XL Objektiv muss unter Einstellungen im Menü Datei aktiviert sein.
    • Schachbrettbilder können in \Users\{Current User}\AppData\Local\Martin\EasySolder\{App. Version}\Chessboardsenthalten sein um das manuelle Fotografieren zu überspringen. Diese Bilder müssen die Bilder einer früheren Kalibrierung sein, die auf demselben Gerät mit denselben Objektiven aufgenommen wurden.
      • Alte Schachbrettbilder können nicht wiederverwendet werden, wenn sich Fokus oder Blende geändert haben.
    • Wenn die Schachbrettbilder manuell in den Ordner aufgenommen wurden, erkennt die Anwendung sie automatisch und die Schaltfläche Kalibrieren wird direkt verwendet.
    • Wenn die oben beschriebene Option nicht verfügbar ist, müssen die Schachbrettbilder manuell mit der Gerätekamera gescannt werden.
    • Beim Scannen der Schachbretter wird die Schaltfläche Erkennen verwendet, dann die Schaltfläche Kalibrieren.
    • Die Taste Löschen löscht die im Ordner gespeicherten Schachbrettbilder.

    # Schritte der Kamerakalibrierung

    Das Kalibrierschachbrettmuster muss auf einer harten und sehr ebenen, nicht biegbaren Oberfläche (z.B. Glas) platziert werden.

    • Vergewissern Sie sich, dass die Einstellungen im Fenster Kamerakalibrierung korrekt sind.
      • Bei Verwendung der Schachbrett-Vorrichtung in der Datei Kalibriervorlage für MaxiBGA XL.png sind die Einstellungen wahrscheinlich bereits auf die richtigen Werte eingestellt.
      • Der Breitenwert sollte gleich der Anzahl der horizontalen Quadrate minus einem 19 sein.
      • Der Höhenwert sollte gleich der Anzahl der vertikalen Felder minus einem 13 sein.
      • Der Wert der Quadratgröße sollte gleich der Seite des Quadrats in Millimetern sein. Bitte stellen Sie sicher, dass dieser Wert so genau wie möglich ist. Zur Messung der Winkelgröße kann ein digitales Messschieberwerkzeug verwendet werden.
      • Für eine bessere Genauigkeit können mehr Quadrate gemessen werden, dann muss der Wert durch die Anzahl der Quadrate geteilt werden.
    • Nehmen Sie die Bilder mit der Taste Erkennen auf.
    • Wenn ein Schachbrett nicht erkannt wird:
      • Stoppen Sie die Erkennung, indem Sie das Kontrollkästchen Kontinuierliche Erkennung deaktivieren.
      • Bewegen Sie das Schachbrett leicht
      • Aktivieren Sie die kontinuierliche Erkennung und drücken Sie erneut Erkennen.
    • Nachdem die Bilder an jeder Position auf der Leiterplatte aufgenommen wurden, klicken Sie auf die Schaltfläche Kalibrieren. Die Kalibrierung dauert einige Minuten.

    Benötigte Bildkategorien

    • Schachbretter so weit wie möglich von der Kamera entfernt, aber immer noch im Fokus.
    • Schachbretter so nah wie möglich an der Kamera, aber immer noch im Fokus.
    • Schachbretter irgendwo zwischen nah und fern von der Kamera
      Für die obigen Kategorien der flachen Bilder ist darauf zu achten, dass der gesamte Bildbereich mit einer Bildsequenz abgedeckt wird.
    • Kippschachbretter (oben, unten, links, rechts)
    • Geneigte Bilder in der Nähe des Bildrandes

    # Kameraausrichtung

    • Setzen Sie das normale BGA-Objektiv ein und befestigen Sie die BGA/QFP 10 Düse.
      • Andere Objektive oder Düsen können für diesen Schritt nicht verwendet werden.
    • Stellen Sie sicher, dass die BGA/QFP 15 Düse in Datei ->Einstellungen deaktiviert ist.
    • Stellen Sie sicher, dass keine Leiterplatte / Board über der Unterheizung installiert ist.
    • Arm-Kalibrierung durchführen
    • Im Ausrichtungsschritt der Kamera muss sich der Blob innerhalb des roten Kreises befinden. Er muss nicht genau in der Kreismitte sein, nur innerhalb des Kreises.
      • Wenn die Ausrichtung gut ist, drücken Sie Fortsetzen.
      • Warten Sie, bis der Rest der Kalibrierschritte abgeschlossen ist, und schließen Sie das Kalibrierfenster.

    # Licht- und Fokuseinstellung

    • BGA/QFP 15 Düse und Maxi BGA XL Objektiv einsetzen
    • BGA/QFP 15 Düse in Datei->Einstellungen aktivieren
    • Öffnen Sie Werkzeuge -> Kamerakalibrierung
    • Stecken Sie die Leiterplatte / Board auf die Unterheizung.
    • Licht auf 0 einstellen
    • Klicken Sie auf Auto Helligkeit und warten Sie, bis es abgeschlossen ist.
    • Wenn das Licht nicht in Ordnung ist, stellen Sie die Blende am Objektiv ein und klicken Sie erneut auf Automatische Helligkeit.
      • Wiederholen Sie diesen Schritt, bis das Licht in Ordnung ist.
      • Bitte verwenden Sie die kleinstmögliche Blende, um den größten Fokusbereich für das Objektiv zu erreichen.
      • Sobald das Licht in Ordnung ist, verwenden Sie eine Feststellschraube für die Kamera, um die Blende zu fixieren und sicherzustellen, dass sie sich nicht versehentlich ändert.
    • Bewegen Sie das Kamerakalibrierungsfenster aus dem Weg und wählen Sie das Hauptfenster aus.
    • Drücken Sie Strg+Umschalt+S, um das Fenster Serielle Steuerung zu öffnen.
    • Aktivieren Sie den Schalter Platziere oben und deaktivieren Sie ihn dann. Die Düse senkt sich. Schließen Sie das Fenster zur Kamerakalibrierung nicht, während die Düse sich unten befindet, da sie den Arm bewegen würde.
    • Stellen Sie den Fokus auf dem Objektiv ein, bis sowohl der Blob als auch die Leiterplatte scharf sind.
    • Sobald der Fokus in Ordnung ist, befestigen Sie ihn mit einer Feststellschraube und verhindern Sie, dass er versehentlich verändert wird.
    • Im Fenster der seriellen Steuerung aktivieren Sie den Schalter Platziere oben, um die Düse anzuheben.
    • Schließen Sie das Fenster Serielle Steuerung.
    • Entfernen Sie die Leiterplatte / Board von der Unterheizung.
    • Wenn das Objektiv keine Feststellschrauben hat, verwenden Sie Klebeband oder Kaptonband, um zu verhindern, dass Blende und Fokus versehentlich verändert werden.

    • Wählen Sie die Dosieroption

    • Verwenden Sie den Dosiergriffel, um den Vorgang abzuschließen.

    # Glossar

    Begriff Erklärung
    Reflow-Löten Das Reflow-Löten ist ein Verfahren, bei dem eine Lotpaste (eine klebrige Mischung aus pulverförmigem Lot und Flussmittel) verwendet wird, um ein oder mehrere elektrische Komponenten vorübergehend an ihren Kontaktflächen zu befestigen, woraufhin die gesamte Baugruppe einer kontrollierten Hitze ausgesetzt wird. Dadurch wird das Lot geschmolzen und die Verbindung dauerhaft verbunden. Die Erwärmung kann durch Durchführen der Baugruppe durch einen Reflow-Ofen oder unter einer Infrarotlampe oder durch Löten einzelner Verbindungen mit einem Heißluftkolben erfolgen.
